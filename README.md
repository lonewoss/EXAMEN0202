# EXAMEN0202

WPF-приложение для расчёта стоимости покупки по размерам и материалу, с формированием квитанций.

## Функции

* Расчёт стоимости по кнопке **«Рассчитать»** с выводом: размер, материал, стоимость. 
* Квитанции по кнопке **«Оформить квитанцию»**: создаётся текстовый документ по образцу и сохраняется в файл. 
* Формируется чек **для каждого вида материала**. 

## Тарифы

* Алюминий: **15.50 руб/м²**
* Пластик: **9.90 руб/м²** 

## Формула

Площадь = `ширина × высота`
Стоимость = `площадь × тариф`

## Проверка ввода

* Пустые поля → расчёт не выполняется, показывается сообщение. 
* Отрицательные числа → расчёт не выполняется, показывается сообщение. 

## Квитанции

* Сохраняются в папку `Receipts` рядом с приложением.
* Имя файла: **Уникальный номер чека_дата_стоимость**. 

## Тестирование (по заданию)

Созданы тест-кейсы на:

1. большие числа (макс. длина поля)
2. отрицательные числа
3. пустые поля
